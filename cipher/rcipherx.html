R-Cipher X

Intro

R Cipher X is a version of the R cipher, designed by me which can have an infinitely long key. The cipher is written in Python, and can be found in the Cipher project. A link to that repository can be found on the Projects page.

Commands/How To Use

A live example can be found, here. R-cipher x has 2 commands. After loading the module, type in rcipherx('encrypt'), to get the prompt to encrypt a message. You have to ener lower-case letters, or elese this will not work. To decrypt a message, use the command rcipherx('decrypt'). Again, you have to eneter lower case letters. If you need more help, please create an issue, in the Cipher repository.
